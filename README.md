# Bubble Soap Shop
An online soap shop using MERN stack (MongoDB, Express, React, Node)

## Front-end
* Framework: React. 
* User interaction: HTML canvas. 
* Animation: GSAP. 

## Back-end
* Framework: Node.js, Express. 
* Database: MongoBD. 
* REASful API.
